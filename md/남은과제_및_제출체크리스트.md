## B. 남은 테스트 과제 (우선순위) — 다음 세션

### P1. 업로드 용량 상한 이진탐색 (보통 E-010 강화) · **이미지 준비 완료**
- 준비됨: `바탕화면\X3\용량경계\` = **06/08/10/12/15/20/24MB.jpg** (+ 원본 35MB는 `X3\H`).
- 방법: `업로드_테스트` 스페이스에 **작은 것부터 한 개씩** 업로드 → **통과↔거부가 뒤바뀌는 지점**이 실제 상한.
  - 이미 알려진: 7.4MB 통과, 35MB 거부, 안내는 "5MB". → 상한은 7.4~35MB 사이.
- 산출: 정확한 상한값 + "안내(5MB)와 실제 상한 불일치" 근거 강화. **업로드는 직접 필요.**

### P2. 비율 변형 개별 확정 (검증 부재 확인) · **이미지 준비 완료**
- 준비됨: `업로드테스트\` = `03_비율_정사각`, `04_비율_16x9`, `05_비율_세로`.
- 각각 단독 업로드 → 통과 시 왜곡/검증 부재(E-009 계열 보강), 거부 시 안내 확인.

### P3. E-008(심각) 크래시 재현조건 좁히기
- 배치 업로드에서 white screen 발생했으나 단일 파일 미재현 → **몇 장부터/어떤 조합에서** 나는지 좁혀 재현절차 확정(심각 결함 방어의 핵심).

### P4. 대형 투어(37씬, H동 층별투어) 로딩·성능·콘솔
- 이번 세션 **카드 클릭 시 회의실&구역(cntId149)이 열리는 진입 이슈**로 미완(→ C절 E-018). 진입 해결 후 37씬 로딩/씬이동/콘솔/프리즈 측정.
