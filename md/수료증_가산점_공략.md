# 수료증 가산점 공략 — 마감(7/20) 전에 최대한 쌓기

> 2026 중부권 SW테스트 경진대회(세종) · 가산점 항목 공략
> 작성 2026-07-08 · **갱신 2026-07-13** · 마감까지 **7일**
> ⚠ 남은 기간이 짧으므로 **즉시 발급형(해외 무료 플랫폼)을 물량 중심**으로, K-MOOC 등 기간제는 **마감 전 이수 가능한 것만** 선별.

---

## 규칙 재확인 (배점표 기준)

| 가산 항목 | 가점 | 상한 |
|---|---|---|
| SW테스팅 자격증 (ISTQB/KSTQB/CSTS 등) | **자격증당 2점** | 명시 없음 |
| **SW 공학·품질관리·테스팅·개발 관련 교육 이수 수료증** | **수료증당 1점** | **명시 없음** |
| SW 관련 학과 재학/수료 | 2점 | — |

> **핵심**: 수료증은 **장당 1점, 상한이 없음** → **관련 분야로 여러 장을 쌓는 게 그대로 순수 가산.**
> 단, 자격 조건은 **"SW 공학·품질관리·테스팅·개발 관련"** → 마케팅·엑셀 같은 건 인정 안 될 수 있으니 **주제를 SW 개발/테스팅/품질/공학으로 한정**할 것.

---

## 전략: "신뢰 코어 + 물량 스택" 2단 구성

1. **신뢰 코어 (한국 공식기관 수료증 2~4장)** — 심사위원이 확실히 인정하는 공신력 있는 것. 대회 주제(테스팅/품질)와 직결되게.
2. **물량 스택 (해외 무료 플랫폼 수료증 여러 장)** — 1~3시간에 1장씩 즉시 발급. 관련 주제로 빠르게 누적.

> ⚠ **주의**: 배점표에 "일부 변동될 수 있음" 문구가 있어 **과도한 저품질 물량은 심사 재량으로 할인될 여지**가 있음. 그래서 **코어(공식)로 신뢰를 깔고**, 스택은 **제목이 SW개발/테스팅/품질로 명확한 것**만 골라 담는 게 안전. 무관해 보이는 강좌는 담지 말 것.

## B. 물량 스택 — 해외 무료 플랫폼 (1~3시간, 즉시 수료증)

전부 **무료 + 자기주도 + 완료 즉시 수료증(PDF, 이름 표기)**. 대회 주제와 맞는 강좌만 추림.

### ⓪ Test Automation University (Applitools) — **테스팅 특화라 관련성 최상**
| 강좌 예시 | 소요 | 관련성 |
|---|---|---|
| Setting a Foundation for Successful Test Automation | ~1.5h | 테스팅 ★★★ |
| Web Element Locator Strategies / Exploring Service APIs through Test Automation | 1~2h | 테스팅 ★★★ |
| Selenium WebDriver / Playwright / Cypress 입문 강좌 다수 | 1~2h | 테스팅 ★★★ |
- 링크: [Test Automation University](https://testautomationu.applitools.com/)
- 특징: 전 강좌 무료, 강좌당 60~90분 위주, 퀴즈 통과 시 **강좌별 수료증(이름 표기 PDF)** 발급. **강좌 제목 자체가 전부 "테스트/자동화"라 심사 관련성 논란이 전혀 없음 → 최우선 스택.**

### ① Great Learning Academy (가장 빠른 스택 — 강좌당 1~3h)
| 강좌 예시 | 소요 | 관련성 |
|---|---|---|
| Software Testing Fundamentals | ~2.25h | 테스팅 ★★★ |
| Introduction to Automation Testing | ~1h | 테스팅 ★★★ |
| SDLC (소프트웨어 개발 생명주기) | ~1h | 개발/공학 ★★ |
| Manual Testing / Selenium 기초 | 1~2h | 테스팅 ★★ |
| SQL / Python 기초 | 1~2h | 개발 ★★ |
- 링크: [Great Learning — Software Testing Fundamentals](https://www.mygreatlearning.com/academy/learn-for-free/courses/software-testing-fundamentals1)
- 특징: 가입 후 평생 접근, 모듈 이수 + 간단 평가 통과 → 수료증. **하루에 2~3장도 가능.**

### ② Simplilearn SkillUp (무료 + 수료증)
| 강좌 예시 | 관련성 |
|---|---|
| Software Testing Course (무료) | 테스팅 ★★★ |
| Introduction to Software Development / Programming | 개발 ★★ |
- 링크: [Simplilearn — Free Software Testing Course](https://www.simplilearn.com/free-software-testing-course-skillup)

### ③ HackerRank Skills Certification (약 1h 시험 → 즉시 인증서)

- 특징: 강의가 아니라 **짧은 테스트 통과 시 인증서** → 실력 있으면 초고속 스택.

## C. 7일 압축 스케줄 (7/13~7/19 · 팀원 각자, **저녁·자투리 시간에만**)

목표: **1인당 수료증 5~8장** (팀 2인이면 10~16장 = 가산 10~16점 잠재). 낮은 제품 테스트에 집중, 수료증은 저녁에.

| 기간 | 할 일 | 산출 |
|---|---|---|
| 7/13(월) | 플랫폼 프로필 **영문/국문 이름을 참가신청서와 일치**하게 세팅 + Test Automation University·Great Learning 가입 → 테스팅 강좌 1장씩 착수 | 즉시 수료증 1~2장 |
| 7/14~7/15 | Great Learning "Software Testing Fundamentals"·"Automation Testing"·SDLC + Simplilearn 테스팅 | 즉시 수료증 3~4장 |
| 7/16~7/17 | HackerRank Problem Solving/SQL 인증 + Test Automation University 추가 강좌 | 수료증 2~3장 |
| 7/18 | (여력 시) K-MOOC/STEP 등 **마감 전 이수 가능한** 코어 강좌 마무리 | 코어 수료증 |
| 7/19 오전 | **모든 수료증 PDF 저장 + 이름·날짜(≤7/20) 확인 + 서류철 정리 → 본 제출에 첨부** | 제출본 완성 |

> 시간이 짧으므로 **즉시 발급형 우선.** 기간제(K-MOOC)는 마감 전 이수 처리 안 되면 미련 없이 스킵.

## D. 제출 전 반드시 체크 (인정 안 되면 0점)

- [ ] **이름 일치**: 수료증의 영문/국문 이름이 참가신청서 이름과 매칭되게 발급 (플랫폼 프로필 이름 미리 세팅)
- [ ] **발급일 ≤ 7/20**: 마감 이후 발급본은 인정 리스크
- [ ] **주제 관련성**: "SW 공학·품질관리·테스팅·개발"에 부합하는 제목인지 (마케팅/오피스 등은 제외)
- [ ] **PDF/이미지 저장**: 캡처 말고 **공식 PDF 다운로드** (검증 URL 있으면 함께)
- [ ] **팀원분도 동일 진행**: 팀 구성원 전원 수료증이 각각 가산 → 팀원도 같이 쌓기
- [ ] **재학증명서(관련학과 2점)**와 함께 "가산점 증빙" 폴더로 묶어 제출

## 요약

- 수료증은 **장당 1점·상한 없음** → **관련 분야로 많이 쌓을수록 이득**.
- **코어 2~4장(K-MOOC·KISA·배움나라 등 공식)**으로 신뢰를 깔고, **Great Learning·Simplilearn·HackerRank로 물량(1인 5~8장)**을 채운다.
- **제목이 SW개발/테스팅/품질로 명확한 것만** 담고, 이름·날짜·PDF를 챙긴다.
- 팀원 2인이 함께하면 **10점 이상 가산도 현실적** → 동점 경쟁에서 결정적.
