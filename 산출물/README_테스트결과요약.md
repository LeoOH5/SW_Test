## 이번 테스트로 확보한 것
- **테스트 케이스 20건** (PASS 7 / FAIL 13) — 기능 인벤토리 전 영역 커버, PASS 포함
- **결함 11건**: 심각 1(E-001), 보통 1(E-002), 경미 7, 개선 2
- **핵심 결함 2건이 득점 앵커**
  - **E-001 (심각)**: 초소형(100×50) 이미지 업로드 시 360 뷰어 전체 검은 화면 → 단일 씬 VT는 투어 전체 표시 불가(Breakdown). `07_포맷.png` 정상 렌더와 대조 명확.
  - **E-002 (보통)**: 잘못된 VT ID 접근 시 관리자 저작 화면·"360이미지 등록" 버튼을 익명 노출(오류처리 부재).
