정상 PASS — 커버리지 증빙

  • 권한/IDOR 차단: cmpnId를 미소속 값(1·2·50·99·150·200·300)으로 바꿔도 code103 "not a member" 차단. 접근된 건 내 소속 회사(100·101)뿐. 인접 ID 99 차단으로 순차 IDOR도 반증.
  • 비인증 접근 차단: 토큰 없이 API 호출 → 401 "token is not valid".
  • 서버측 입력검증 존재: 로그인 빈값/형식오류/비번 길이 위반 → 400(class-validator).
  • 테스트 스페이스 비노출: 두 스페이스 모두 active=false (실서비스 메인 미노출) 확인.
